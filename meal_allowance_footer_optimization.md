## 🔧 实施步骤

### 步骤 1：移除 table-dark 类

#### 修改前
```html
<tfoot class="table-dark">
    <tr>
        <!-- 合计内容 -->
    </tr>
</tfoot>
```

#### 修改后
```html
<tfoot class="table-footer-custom">
    <tr>
        <!-- 合计内容 -->
    </tr>
</tfoot>
```

**说明**：将 `table-dark` 类替换为自定义的 `table-footer-custom` 类

---

### 步骤 2：添加浅灰色背景 CSS 样式

在"特殊功能样式"区域添加以下 CSS：

```css
/* 表格合计行浅灰背景样式 */
.table-footer-custom {
    background-color: #f8f9fa !important;
    border-top: 2px solid #dee2e6;
}

.table-footer-custom td,
.table-footer-custom strong {
    color: #212529;
}

.table-footer-custom .text-info {
    color: #5a75f9 !important;
}

.table-footer-custom .text-success {
    color: #00c569 !important;
}

.table-footer-custom .amount-positive {
    color: #f46363 !important;
}
```

## 🔍 技术细节

### CSS 优先级处理
```css
/* 使用 !important 确保样式覆盖 Bootstrap 默认样式 */
.table-footer-custom {
    background-color: #f8f9fa !important;  /* 强制覆盖 */
}
```

### 文字颜色针对性设置
```css
/* 基础文字颜色 */
.table-footer-custom td,
.table-footer-custom strong {
    color: #212529;  /* 深灰色，保证可读性 */

/* 特殊类型文字颜色 */
```
