# 酒店统计管理页面 Espire 设计风格优化报告

## 任务概述

对 `/admin/hotel_statistics_admin.php` 页面进行全面的视觉优化，采用 Espire 设计系统的颜色方案、组件样式和响应式设计规范。

## 优化范围

1. **颜色系统** - 实现完整的 Espire CSS 变量系统
2. **卡片设计** - 统一的卡片头部、阴影、边框样式
3. **表格优化** - 改进表头、行间距、颜色搭配
4. **按钮和表单** - 统一的控件样式和交互效果
5. **响应式设计** - 优化多断点的布局适配
6. **菜单激活** - 添加菜单标记变量，确保页面在正确菜单下

## 修改详情

### 1. CSS 颜色系统优化

**新增：Espire 颜色变量**
```css
:root {
    --espire-primary: #11a1fd;
    --espire-success: #00c569;
    --espire-info: #5a75f9;
    --espire-warning: #ffc833;
    --espire-danger: #f46363;
    --espire-bg-light: #f8f9fa;
    --espire-border: #e9ecef;
    --espire-text-dark: #1a1a1a;
    --espire-text-gray: #6c757d;
}
```

### 2. 卡片样式优化

**统计卡片（stats-card）**：
- 添加顶部彩色下边框（3px）
- 卡片头部使用浅色背景和深色文字
- 卡片悬停效果：向上平移 2px，增加阴影
- 六张统计卡片的下边框颜色分别为：蓝、黄、红、蓝、绿、红

**普通卡片**：
- 渐变头部背景：从 `#11a1fd` 到 `#0d7fd8`
- 白色文字，无下边框
- 圆角 0.75rem，阴影 0 0.25rem 0.5rem

### 3. 表格样式优化

**表头样式**：
- 深灰色背景 (`#2d3436`)
- 白色文字，粗体
- 主边框颜色从 `#dee2e6` 改为 `var(--espire-primary)`

**行样式**：
- 悬停背景色改为浅蓝色透明 `rgba(17, 161, 253, 0.05)`
- 添加过渡动画 `all 0.2s ease`
- 移除阴影效果

**表格列增强**：
- 为每个表头添加相关 icon（日期、酒店、人员等）
- 表头 icon 右边距 `me-1`

### 4. 按钮样式优化

**按钮通用设置**：
- 圆角 0.5rem，无边框
- 字重 600，字体大小 0.9375rem
- 过渡动画 `all 0.3s ease`

**按钮类型**：
- `btn-primary`：蓝色背景，悬停时亮度变高
- `btn-secondary`：灰色背景
- `btn-outline-secondary`：无背景，虚线边框
- `btn-success`：绿色背景
- `btn-danger`：红色背景

**按钮 icon**：
- 右边距 `me-1`（margin-end 0.25rem）

### 5. 表单控件优化

**输入框和下拉菜单**：
- 边框颜色 `var(--espire-border)`
- 焦点时边框色变为 `var(--espire-primary)`
- 焦点时阴影 `0 0 0 0.2rem rgba(17, 161, 253, 0.15)`
- 圆角 0.5rem
- 字体大小 0.9375rem

**标签样式**：
- 字重 600，颜色 `var(--espire-text-dark)`
- 下边距 0.5rem

### 6. 菜单激活修复

**在 PHP 代码中添加**（第 757-759 行）：
```php
// 为了让菜单能够识别这个页面，设置菜单标记变量
$current_editing_page = 'hotel_statistics';
```

这样当用户访问此页面时，左侧菜单会自动在正确的位置显示激活状态。

### 7. 卡片头部优化

所有卡片头部都进行了如下优化：

**筛选卡片**：
- 标题前增加 icon：`<i class="bi bi-funnel-fill me-2"></i>`
- 表单标签前增加 icon（文件夹、建筑物）
- 按钮增加 icon 和更好的间距

**统计卡片**：
- 添加卡片标题在头部（如"总预订"、"房间数"）
- Icon 字体大小增加到 2.5rem
- 卡片按照不同颜色进行视觉区分

**表格卡片**：
- 标题增加对应 icon
- 表格标题栏增加列 icon（日期、人员、建筑、房间等）

### 8. 响应式设计优化

**三个主要断点**：

#### 1200px 以下：
- 表格字体大小 0.8rem
- 表格行高间距减小
- 统计卡片底部边距减少

#### 768px 以下（平板）：
- 容器内边距 1rem
- 表格字体大小 0.75rem
- 统计网格纵向排列
- 表单控件字体大小减小
- 卡片头部和内容边距减少

#### 576px 以下（手机）：
- 容器内边距 0.75rem
- 表格字体大小 0.7rem
- 统计卡片底部边距最小化
- 所有间距和字体进一步减小

### 9. DataTables 样式优化

- 分页按钮添加 Espire 风格
- 搜索框圆角和焦点效果优化
- 分页按钮悬停状态：浅背景 + 蓝色边框 + 蓝色文字
- 当前页按钮：蓝色背景 + 白色文字

### 10. 模态框和提示框优化

**模态框**：
- 头部使用渐变背景（同卡片）
- 标题前可添加 icon
- 关闭按钮使用反相颜色
- 底部边框替代为 border-top
