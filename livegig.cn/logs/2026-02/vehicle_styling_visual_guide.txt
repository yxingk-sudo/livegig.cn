🎯 样式应用层级 (Styling Hierarchy)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

每辆车的样式应用于三个层级，形成递进式的视觉强调：

📍 第1层级 - 表格行级别 (Highest Priority)
   ├─ 类名: .vehicle-trip-group-N
   ├─ 样式: 4px 左边框 + 淡色背景
   ├─ 效果: 快速扫视表格时立即识别车辆
   └─ 示例: 整行都采用该车的颜色
   
   样式代码:
   .vehicle-trip-group-1 {
       background-color: rgba(17, 161, 253, 0.05);  ← 5% 透明度蓝色
       border-left: 4px solid #11a1fd;              ← 4px 蓝色边框
   }

📍 第2层级 - 行程卡片级别 (Medium Priority)
   ├─ 类名: .vehicle-trip-group-N .trip-item
   ├─ 样式: 3px 左边框 + 中等透明度背景
   ├─ 效果: 强调单个行程信息所属车辆
   └─ 示例: 【行程1】、【行程2】 都采用该车的颜色
   
   样式代码:
   .vehicle-trip-group-1 .trip-item {
       border-left: 3px solid #11a1fd;              ← 3px 蓝色边框
       background-color: rgba(17, 161, 253, 0.06);  ← 6% 透明度蓝色
   }

📍 第3层级 - 乘客组级别 (Lowest Priority)
   ├─ 类名: .vehicle-trip-group-N .trip-passengers-group
   ├─ 样式: 3px 左边框 + 浅色背景
   ├─ 效果: 乘客信息与行程关联同一车辆
   └─ 示例: 【乘客】 也采用该车的颜色
   
   样式代码:
   .vehicle-trip-group-1 .trip-passengers-group {
       border-left: 3px solid #11a1fd;              ← 3px 蓝色边框
       background-color: rgba(17, 161, 253, 0.05);  ← 5% 透明度蓝色
   }

✨ 视觉设计原则 (Design Principles)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ 颜色可区分性 (Color Discernibility)
  - 6 种颜色采用对比度设计，相邻车辆颜色差异明显
  - 蓝、绿、橙、红、紫、青 —— 色系完全不同
  - 支持色盲用户：使用不同饱和度和色调区分

✓ 透明度美学 (Transparency Aesthetics)
  - 避免使用完全不透明的背景，保留文本清晰度
  - 5-8% 透明度范围内，既能提示颜色分组，又不影响阅读
  - 与 Espire 设计系统的「玻璃态」风格一致

✓ 边框层级 (Border Hierarchy)
  - 4px 表行左边框：距离最远，快速识别
  - 3px 内部元素边框：距离近，精细对应
  - 逐级递减，形成视觉焦点引导

✓ 一致性 (Consistency)
  - 同一辆车的所有行（空闲行或行程行）使用相同颜色
  - 行内的三个层级元素（行程卡、乘客组）使用相同颜色系
  - 跨页面加载时，同一辆车始终保持相同颜色

✓ 易用性 (Usability)
  - 无需交互操作即可识别车辆分组（免点击）
  - 适配所有屏幕尺寸（响应式）
  - 打印友好（低透明度，文本清晰）

✓ 桌面（≥1200px）     - 完整 4px 左边框，可见所有细节
✓ 平板（992-1200px）  - 左边框保持，行间距调整
✓ 移动（768-992px）   - 左边框宽度保持，字体自适应
✓ 小屏（<768px）      - 左边框保持，表格可横向滚动
