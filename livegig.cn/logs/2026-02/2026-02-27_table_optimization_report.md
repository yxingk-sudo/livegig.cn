# 酒店统计页面 - 表格深度优化报告

## 优化范围

对 `/admin/hotel_statistics_admin.php` 中的两个核心表格进行深度视觉优化：
1. **每日入住统计表**（dailyTable）- 5列表格
2. **入住人员详情表**（personnelTable）- 7列表格

## 优化内容详解

### 1. CSS 颜色与样式系统

#### 表头优化
- 背景色：深灰色 `#2d3436`
- 字色：纯白 `#ffffff`
- 字重：600（粗体）
- 下边框：2px Espire 主色 `var(--espire-primary)`

#### 行悬停效果
- 背景色：浅蓝透明 `rgba(17, 161, 253, 0.08)`
- 过渡时间：0.2s ease
- 垂直移动：向上 1px

### 2. 每日入住统计表（dailyTable）优化

#### 列特定样式

**日期列（date-info）**
- 字族：Courier New（等宽字体，便于对齐）
- 字重：600
- 显示：日期 + 星期
- 内边距：8px 10px
- 颜色：主色

**酒店信息列（hotel-info）**
- 内边距：8px 10px
- 子标题颜色：灰色
- 字体大小降低为 0.75rem
- 弹性换行支持

**入住统计列（stats-info）**
- 居中对齐
- 显示：人数 + icon
- 内边距：8px 6px

**房间统计列（rooms-info）**
- Flexbox 布局
- 展示间数和房晚数
- 响应式排列

**房型列（room-types-col）**
- 徽章显示（badge）
- 颜色区分：双床房(蓝)、大床房(绿)、套房(黄)、其他(灰)
- 响应式换行

#### 列宽设置（含最小宽度）
| 列名 | 宽度 | 最小宽度 |
|------|------|---------|
| date-col | 18% | 140px |
| hotel-col | 25% | 180px |
| stats-col | 15% | 100px |
| rooms-col | 20% | 140px |
| room-types-col | 22% | 150px |

### 3. 入住人员详情表（personnelTable）优化

#### 列特定样式

**序号列（sequence-col）**
- 宽度：60px，可缩放
- 显示：带渐变背景的圆形徽章
- 背景：渐变蓝色 `linear-gradient(135deg, #11a1fd, #0d7fd8)`
- 阴影：0 2px 4px rgba(17, 161, 253, 0.3)
- 字体：粗体，白色，居中

**人员信息列（personnel-info）**
- 共享房间：显示"X人共享"标签
- 独立房间：单人显示
- 名字最大宽度：150px，超长省略
- 字重：600

**酒店信息列（hotel-info）**
- 显示酒店名 + 项目名
- 项目名 0.75rem 灰色
- 最大宽度：150px

**入住日期列（dates-info）**
- 显示入住 + 退房日期
- 日期使用 Courier New 等宽字体
- 显示停留天数
- 内边距：8px 10px

**房型信息列（room-info）**
- 显示房型徽章（蓝/绿）
- 显示共享/独立标签
- 居中对齐

**统计信息列（stats-info）**
- 显示：房间数、房晚数、人数
- 网格布局，3 个统计项
- 颜色区分（蓝、黄、绿）

**特殊要求列（requirements-col）**
- 最多显示 20 个字符，超长省略...
- Hover 时变色（蓝色）
- Tooltip 显示完整内容

#### 列宽设置（含最小宽度）
| 列名 | 宽度 | 最小宽度 |
|------|------|---------|
| sequence-col | 5% | 60px |
| personnel-col | 16% | 150px |
| hotel-col | 18% | 160px |
| dates-col | 18% | 160px |
| room-col | 13% | 120px |
| stats-col | 12% | 100px |
| requirements-col | 18% | 150px |

### 4. 徽章和统计项优化

#### 徽章（.badge）
- 圆角：0.35rem
- 字重：600
- 内边距：0.35rem 0.65rem
- 悬停效果：向上 1px + 阴影增强

#### 房型徽章（.room-type-badge）
- 字体大小：0.75rem
- 内边距：3px 8px
- 圆角：0.35rem
- 悬停效果：向上 1px + 阴影

#### 统计项（.stat-item）
- Flexbox 垂直排列
- 间距：2px
- 值字重：700
- 标签字色：灰色

#### 统计网格（.stats-grid）
- Flexbox 水平排列
- 间距：8px
- 排列方式：center

### 5. 响应式设计优化

#### 1200px 及以上（桌面）
- 表格字体：0.85rem
- 行高：正常
- 列宽：全尺寸
- 显示：完整信息

#### 768px - 1200px（平板）
- 表格字体：0.8rem
- 行内边距：6px 4px
- 列最小宽度：减 20%
- 隐藏非关键信息

#### 576px - 768px（小平板）
- 表格字体：0.75rem
- 行内边距：4px 2px
- 列最小宽度：减 40%
- 统计项纵向排列

#### 576px 以下（手机）
- 表格字体：0.7rem
- 行内边距：3px 1px
- 列最小宽度：减 50%
- 名字最大宽度：70px
- 统计值：0.75rem
- 统计标签：0.55rem

### 6. 颜色系统应用

**表头颜色**

**文本颜色**

- 统计值：根据类型使用主题色

## 技术细节

### 列宽策略
- 使用百分比宽度确保响应式
- 设置 min-width 确保内容不拥挤
- 使用 max-width 和 text-overflow: ellipsis 处理长文本

### 颜色管理
- 所有颜色使用 CSS 变量
- 便于后续全局修改
- 与整个系统的设计保持一致

### 悬停效果
- 使用 transform: translateY(-1px) 模拟浮起
- 添加 box-shadow 增强立体感
- 过渡动画 0.2s ease 提供平滑感受

### 响应式策略
- 1200px 删除不必要空间
- 768px 调整关键信息优先级
- 576px 最小化空间占用
- 保持核心数据始终可见
