## 2. 出行安排列文字颜色优化

### 优化目标
- 增强行程时间与路线信息的对比度
- 避免颜色重叠导致难以阅读
- 确保与背景色有充分的视觉区分

### 具体修改

#### 行程时间信息 (.trip-time)
```css
旧方案: color: (继承，文本颜色由背景决定)
       font-size: 0.8rem

新方案: color: var(--espire-primary)  ← 主色蓝色，对比度强
       font-size: 0.9rem             ← 字号更大
       margin-bottom: 4px → 6px      ← 间距更大
```

#### 行程路线信息 (.trip-route)
```css
旧方案: color: text-muted (灰色，容易与背景融合)
       font-size: 0.75rem

新方案: color: var(--espire-text-dark)  ← 深色，对比度强
       font-size: 0.9rem                ← 字号更大
       line-height: 1.3 → 1.4           ← 行距更大
```

#### 出发地/目的地颜色处理
```css
旧方案: .text-success (绿色) 和 .text-danger (红色)
       可能与背景色冲突

新方案: 新方案中，对于有车辆样式的行，
       出发地和目的地都改为 var(--espire-text-dark)
       
       /* 示例：1号车蓝色 */
       .vehicle-trip-group-1 .trip-route .text-success {
           color: var(--espire-text-dark) !important;
       }
```

### 对比度改进效果

| 元素 | 旧方案对比度 | 新方案对比度 | 改进 |
|------|------------|------------|------|
| 行程时间 (时间 + 背景) | 中等 | 强 (绿色) | ⬆⬆⬆ |
| 行程路线 (文字 + 背景) | 弱 (灰色) | 强 (深色) | ⬆⬆⬆⬆ |
| 整体可读性 | ⭐⭐ | ⭐⭐⭐⭐⭐ | 显著提升 |

## 3. 新车辆级视觉样式方案

### 优化目标
- 用更加专业、优雅的视觉方案替代纯色背景方案
- 引入渐变背景提升视觉层次感
- 增加边框细节设计以强调车辆分组
- 避免与其他表格元素颜色冲突

### 旧方案问题
- 使用纯色 RGBA 背景，显得单调
- 颜色对比度高但视觉效果不够柔和
- 背景颜色容易与文本冲突
- 缺乏视觉层次感

### 新方案特点

#### 1. 渐变背景设计
```css
/* 表行背景：从左到右的微妙渐变 */
.vehicle-trip-group-1 {
    background: linear-gradient(90deg, #f0f7ff 0%, #ffffff 100%);
}

/* 行程行：更深的渐变 */
.vehicle-trip-group-1.vehicle-trips-row {
    background: linear-gradient(90deg, #e8f1ff 0%, #ffffff 100%);
}

/* 行程卡片：对角线渐变 */
.vehicle-trip-group-1 .trip-item {
    background: linear-gradient(135deg, rgba(0, 102, 204, 0.08) 0%, transparent 100%) !important;
}

/* 乘客组：同样的对角线渐变 */
.vehicle-trip-group-1 .trip-passengers-group {
    background: linear-gradient(135deg, rgba(0, 102, 204, 0.08) 0%, transparent 100%);
}
```

**好处**:
- 从浅色过渡到透明，视觉更自然
- 保留了白色底部，文字对比度好
- 具有层次感和深度感

#### 2. 精妙边框设计
```css
/* 表行左边框：5px，作为主视觉标识 */
.vehicle-trip-group-N {
    border-left: 5px solid #RRGGBB;
}

/* 顶部细边框：1px，细微的视觉分割 */
.vehicle-trip-group-N {
    border-top: 1px solid rgba(R, G, B, 0.15);
}

/* 行程卡片：4px，强调内部结构 */
.vehicle-trip-group-N .trip-item {
    border-left: 4px solid #RRGGBB !important;
}

/* 乘客组：4px，保持一致性 */
.vehicle-trip-group-N .trip-passengers-group {
    border-left: 4px solid #RRGGBB;
}
```

**边框层级**:
- 5px (表行) > 4px (内部元素) > 1px (细分割)
- 形成清晰的视觉层级

#### 3. 文字颜色统一
```css
/* 新方案中，所有行程路由文字都统一为深色 */
.vehicle-trip-group-N .trip-item {
    color: var(--espire-text-dark);  /* 统一深灰/黑色 */
}

.vehicle-trip-group-N .trip-route .text-success {
    color: var(--espire-text-dark) !important;  /* 覆盖绿色 */
}

.vehicle-trip-group-N .trip-route .text-danger {
    color: var(--espire-text-dark) !important;  /* 覆盖红色 */
}

.vehicle-trip-group-N .trip-passengers-group {
    color: var(--espire-text-dark);
}
```

**原因**:
- 绿色/红色与各车颜色可能冲突
- 深色文字对所有背景都有好对比度
- 使表格显得更加统一专业
