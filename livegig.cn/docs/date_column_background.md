# 日期大列背景色区分 - 样式说明

## 功能概述

为前台报餐管理页面的每个日期大列添加不同的背景颜色，通过色彩区分使日期边界更加清晰，提升视觉层次感和可读性。

## 设计方案

### 配色方案（增强版 - 更明显）

采用**四色循环渐变**设计，每 4 天为一个循环周期，使用更高的透明度值使颜色更加明显：

| 序号 | 颜色主题 | 表头渐变 | 表体背景 | 适用日期 |
|------|---------|---------|---------|---------|
| **第 1 天** | 🔵 蓝色系 | `rgba(52, 152, 219, 0.15)` → `rgba(41, 128, 185, 0.10)` | `rgba(52, 152, 219, 0.08)` | 1, 5, 9, 13... |
| **第 2 天** | 🟢 绿色系 | `rgba(46, 204, 113, 0.15)` → `rgba(39, 174, 96, 0.10)` | `rgba(46, 204, 113, 0.08)` | 2, 6, 10, 14... |
| **第 3 天** | 🟠 橙色系 | `rgba(243, 156, 18, 0.15)` → `rgba(211, 84, 0, 0.10)` | `rgba(243, 156, 18, 0.08)` | 3, 7, 11, 15... |
| **第 4 天** | 🟣 紫色系 | `rgba(155, 89, 182, 0.15)` → `rgba(142, 68, 173, 0.10)` | `rgba(155, 89, 182, 0.08)` | 4, 8, 12, 16... |

**调整说明：**
- ✅ 表头透明度从 0.08/0.05 提升到 **0.15/0.10**（提升约 2 倍）
- ✅ 表体透明度从 0.03 提升到 **0.08**（提升约 2.7 倍）
- ✅ 色彩更加鲜艳明亮，视觉区分度更高
- ✅ 保持柔和不刺眼，同时确保清晰可辨

### CSS 选择器规则

使用 `:nth-child(4n+x)` 选择器实现循环：

```css
/* 表头背景 */
.date-group-header:nth-child(4n+1) { /* 蓝色系 */ }
.date-group-header:nth-child(4n+2) { /* 绿色系 */ }
.date-group-header:nth-child(4n+3) { /* 橙色系 */ }
.date-group-header:nth-child(4n+4) { /* 紫色系 */ }

/* 表体单元格背景 */
tbody tr td:nth-child(4n+1) { /* 蓝色系 */ }
tbody tr td:nth-child(4n+2) { /* 绿色系 */ }
tbody tr td:nth-child(4n+3) { /* 橙色系 */ }
tbody tr td:nth-child(4n+4) { /* 紫色系 */ }
```

## 视觉效果

### 表头样式

每个日期大列的表头使用**渐变背景**：
- 起始颜色：稍深的色调（透明度 0.08）
- 结束颜色：稍浅的色调（透明度 0.05）
- 渐变方向：135 度对角线渐变

示例：
```css
background: linear-gradient(
    135deg, 
    rgba(17, 161, 253, 0.08) 0%, 
    rgba(13, 138, 230, 0.05) 100%
);
```

### 表体样式

每个日期对应的所有单元格使用**统一淡色背景**：
- 颜色：与表头同色系
- 透明度：0.03（更淡，避免干扰内容）
- 范围：包括该日期的所有餐类型子列

示例：
```css
background-color: rgba(17, 161, 253, 0.03);
```

## 设计原则

### 1. **适度区分**
- ✅ 使用低透明度（0.03-0.08），避免过于鲜艳
- ✅ 保持可读性，不影响文字和复选框
- ✅ 色彩柔和，不刺眼

### 2. **循环规律**
- ✅ 4 天一个循环，形成视觉节奏
- ✅ 易于识别和记忆
- ✅ 符合"周"的概念（虽然不是 7 天）

### 3. **色彩语义**
- 🔵 蓝色：清新、专业
- 🟢 绿色：健康、安全
- 🟠 橙色：活力、温暖
- 🟣 紫色：优雅、独特

### 4. **兼容性**
- ✅ 保留原有边框样式
- ✅ 不影响禁用状态（`.disabled` 优先级更高）
- ✅ 与固定列样式兼容

## 视觉层次

从强到弱的视觉提示：

1. **最强提示**：3px 蓝色粗边框（日期分组边界）
2. **中等提示**：渐变背景色（日期大列区域）
3. **弱提示**：淡色背景（表体单元格）
4. **最弱提示**：1px 灰色细边框（餐类型子列）

这种多层次的设计确保：
- ✅ 日期边界清晰可辨
- ✅ 整体视觉和谐统一
- ✅ 不会喧宾夺主，影响数据阅读

## 技术实现

### 文件位置
- `/www/wwwroot/livegig.cn/user/meal_management.php`

### CSS 代码位置
- 行号：约 205-250（在 `<style>` 标签内）

### 关键代码

#### 表头渐变背景
```css
.date-group-header:nth-child(4n+1) {
    background: linear-gradient(135deg, rgba(17, 161, 253, 0.08) 0%, rgba(13, 138, 230, 0.05) 100%) !important;
}
```

#### 表体淡色背景
```css
tbody tr td:nth-child(4n+1) {
    background-color: rgba(17, 161, 253, 0.03);
}
```

## 自定义配色

如需修改配色方案，可以调整以下参数：

### 修改颜色
```css
/* 蓝色系改为红色系 */
.date-group-header:nth-child(4n+1) {
    background: linear-gradient(135deg, rgba(255, 0, 0, 0.08) 0%, rgba(200, 0, 0, 0.05) 100%) !important;
}

tbody tr td:nth-child(4n+1) {
    background-color: rgba(255, 0, 0, 0.03);
}
```

### 修改循环周期
```css
/* 改为 3 天一个循环 */
.date-group-header:nth-child(3n+1) { /* ... */ }
.date-group-header:nth-child(3n+2) { /* ... */ }
.date-group-header:nth-child(3n+3) { /* ... */ }
```

### 修改透明度
- 表头：调整 `0.08` 和 `0.05` 的值
- 表体：调整 `0.03` 的值
- 范围建议：0.02-0.10

## 响应式支持

背景色样式完全支持响应式设计：
- ✅ 在小屏幕上依然生效
- ✅ 滚动时背景色跟随列移动
- ✅ 触摸设备上色彩表现一致

## 浏览器兼容性

支持所有现代浏览器：
- ✅ Chrome 90+
- ✅ Firefox 88+
- ✅ Safari 14+
- ✅ Edge 90+

降级方案：
- 不支持渐变的浏览器显示纯色背景
- 不支持 `:nth-child` 的浏览器显示默认白色背景

## 注意事项

1. **不要移除 `!important`**
   - 表头背景的 `!important` 是必须的
   - 用于覆盖 Bootstrap 的默认样式

2. **保持透明度一致**
   - 表头：0.08 → 0.05
   - 表体：0.03
   - 修改时需保持这个比例

3. **考虑打印样式**
   - 打印时可能需要移除背景色
   - 可在 `@media print` 中覆盖

4. **性能优化**
   - 使用 CSS 选择器而非 JavaScript
   - 浏览器原生支持，性能最优
