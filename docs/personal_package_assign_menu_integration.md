
## 注意事项

### 开发注意

1. **页面变量设置**：
   ```php
   // personal_package_assign.php 中必须设置
   $active_page = 'personal_package_assign';
   ```

2. **权限控制**：
   - 确保只有项目管理员可以访问此页面
   - 通过 `$_SESSION['project_id']` 隔离数据

3. **样式依赖**：
   - 依赖 Bootstrap 5.3+ CSS
   - 依赖 Bootstrap Icons 1.10+
   - 依赖 Espire设计系统

### 使用注意

1. **访问权限**：
   - 只有登录用户才能看到菜单
   - 需要项目访问权限

2. **数据隔离**：
   - 每个项目只能看到自己的数据
   - 通过 session 中的 project_id 区分

3. **浏览器缓存**：
   - 如果菜单未立即更新，清除浏览器缓存
   - 或使用 Ctrl+F5 强制刷新
